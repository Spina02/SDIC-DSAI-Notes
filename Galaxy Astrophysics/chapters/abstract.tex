\section*{Preface}

This course provides a comprehensive overview of galaxy astrophysics, structured as follows:

\begin{itemize}
    \item \textbf{Fundamentals} (approximately 10 hours)
    \begin{itemize}
        \item Stars
    \end{itemize}
    \item \textbf{Phenomenology} (approximately 25 hours)
    \item \textbf{Dynamics of Non-collisional Systems}
    \begin{itemize}
        \item Elliptical galaxies
        \item Galaxy clusters
    \end{itemize}
\end{itemize}

\vspace{2em}

\textbf{Reference Textbooks:}

\begin{itemize}
    \item \textbf{Introductory}
    \begin{itemize}
        \item Karttunen et al., ``Fundamental Astronomy''
        \item Schneider, P. (2015), ``Extragalactic Astronomy and Cosmology''
    \end{itemize}

    \item \textbf{Advanced}
    \begin{itemize}
        \item Binney, J. \& Merrifield, M., ``Galactic Astronomy''
        \item Binney, J. \& Tremaine, S., ``Galactic Dynamics''
    \end{itemize}
\end{itemize}
